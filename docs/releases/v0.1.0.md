# v0.1.0

## Summary

`ca.jwsm.railroader.api` v0.1.0 is the first public platform release for the new Railroader mod API and host.

This release establishes the core API shape, host bootstrap, and the first real game-facing integration paths so consumer mods can start depending on shared services instead of patching Railroader independently.

## Included in this release

- shared API contracts for `ui`, `trains`, `orders`, `persistence`, and early `world` integration
- a host mod that owns core Harmony patch authority for the first migrated systems
- typed save lifecycle and mod-scoped persistence support
- train/control/UI seams used by early consumer mods
- early world layout and external asset-store support for map package integration

## Current status

This is an early but usable platform checkpoint.

Working areas include:
- host bootstrap and service registry
- save lifecycle observation
- coupler/train event surfaces
- HUD/control integration seams
- early world layout submission and `/MapMods` asset-store registration

Known incomplete areas include:
- turntable realization for map content
- a fuller public map/world API
- broader gameplay domain coverage beyond the first migrated systems

## Intended audience

This release is aimed at active mod development, migration, and testing. It is a foundation release, not a final stability promise.
